# GetBrowserAnalyticsQueryParamInterval

The interval to retrieve analytics for. Takes precedence over start and end.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `Oneh`        | 1h            |
| `TwentyFourh` | 24h           |
| `Sevend`      | 7d            |
| `Thirtyd`     | 30d           |
| `Ninetyd`     | 90d           |
| `Ytd`         | ytd           |
| `Oney`        | 1y            |
| `All`         | all           |
